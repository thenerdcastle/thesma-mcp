# Thesma MCP Server

MCP server that gives AI assistants native access to Thesma's SEC EDGAR data.

## Setup

```bash
pip install thesma-mcp
```

Set your API key:

```bash
export THESMA_API_KEY=your_key_here
```

## Usage

```bash
thesma-mcp
```

See [ARCHITECTURE.md](ARCHITECTURE.md) for details.
